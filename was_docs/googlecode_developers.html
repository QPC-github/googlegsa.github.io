  • upload.py works the same.

  • Send your github username to plexi@google.com.
  • Wait for confirmation.
  • Change git remote repository to those listed in available_repositories with this command: 
                git remote set-url origin https://github.com/googlegsa/<REPO_NAME>.git
  • Now 'git push' will push your changes to github instead of google code site.
